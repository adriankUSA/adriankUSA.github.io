Project Overview

RadiateTPS - Bringing OpenTPS to the Web

Project Overview

Project Name: RadiateTPS

Purpose: This project informs users about Treatment Planning Systems (TPS) used in radiation therapy. It also introduces my personal undergraduate research project with Dr. Yan - RadiateTPS - which adapts OpenTPS into a browser-based, educational and exploratory platform.

Intended Users:

  • Medical and Computer Science/Bioinformatics students
  • Radiation therapy professionals
  • Researchers and educators
  • General public interested in TPS

Content Overview: Explanations of TPS, RadiateTPS Photo Gallery, technical breakdowns, and a contact page.
