Technical Details

RadiateTPS is powered by a combination of web technologies and DICOM handling libraries. It takes advantage of OpenTPS's Python-based dose computation capabilities and modern frontend frameworks to deliver a responsive and educational experience.

Server-Side vs. Browser Rendering

Server-side processing includes DICOM parsing, dose calculation, and ROI interpretation using OpenTPS, Matplotlib, numpy, and Flask in Python. The browser handles interactive visualizations and UI rendering with JavaScript and Plotly.

Tech Stack, Libraries, and Tools
  • Frontend: HTML, CSS, JavaScript
  • Backend: Flask (Python), OpenTPS core components
  • Data: DICOM, RT Structure Sets, CT images
  • Tools: pydicom, numpy, matplotlib, Plotly
Workflow

The user loads or selects a dataset → datasets (DICOM files) are loaded and parsed for correct files (radiation therapy structs/RTStructs) → Regions of Interests (ROIS) are displayed → user selects ROI/dose parameters → receives processed dose visualization in-browser.
