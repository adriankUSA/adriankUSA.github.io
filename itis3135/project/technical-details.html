Technical Details

RadiateTPS is powered by a combination of web technologies and DICOM handling libraries. It takes advantage of OpenTPS's Python-based dose computation capabilities and modern frontend frameworks to deliver a responsive and educational experience.

Tech Stack

  • Frontend: HTML5, CSS3, JavaScript, React.js (planned)
  • Backend: Flask (Python), OpenTPS core components
  • Data: DICOM, RT Structure Sets, CT images
  • Tools: pydicom, numpy, matplotlib, Plotly

Workflow Overview

The user loads a dataset → selects ROI/dose parameters → receives processed dose visualization in-browser.
